# Task
请对我提供的5只股票进行深度复盘与前瞻性分析，并建立一套量化评分体系进行重新评级。

# Input Data

# Analysis Framework
请严格按照以下步骤进行分析：

1. **题材与赛道定位**：
   - 识别每只股票所属的核心题材、行业赛道及当前市场热点关联度。
   
2. **过去上涨逻辑归因（回溯过去3个月）**：
   - 分析过去一段时间股价上涨的核心驱动力（是业绩驱动、政策利好、资金炒作、还是困境反转？）。
   - 区分“实质性利好”与“情绪性博弈”。

3. **逻辑持续性验证（展望未来1-3个月）**：
   - 判断上述上涨逻辑是否依然成立？
   - 是否存在逻辑证伪的风险？（如：利好兑现、估值过高、宏观环境变化）。
   - 未来是否有新的催化剂？

4. **综合评分体系（满分100分）**：
   请基于以下权重计算得分：
   - **基本面健康度 (30%)**：营收/利润增长、估值合理性、现金流。
   - **逻辑确定性 (30%)**：过去逻辑的延续性及新逻辑的强度。
   - **资金与技术面 (20%)**：主力资金流向、筹码结构、趋势形态。
   - **风险收益比 (20%)**：下行空间 vs 上行潜力。
