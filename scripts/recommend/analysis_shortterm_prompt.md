# 分析流程（必须严格执行）

你必须按以下 **四个步骤**分析。

---

# STEP 1：市场环境判断

先判断当前市场 **情绪周期**：

可能的阶段：

- 启动期
- 主升期
- 高潮期
- 退潮期

需要参考：

- 涨停家数
- 跌停家数
- 成交量
- 涨跌家数比
- 龙头表现

输出一句结论：

例如：

市场情绪：主升期（赚钱效应较强）

---

# STEP 2：板块分析（最重要）

短线 **先看板块，再看个股**。

分析每只股票所属板块：

### 板块强度（20分）

根据：

- 板块涨幅排名
- 板块成交额
- 市场关注度

评分规则：

板块涨幅前5 → 20  
前10 → 15  
前20 → 10  
其他 → 0-8

---

### 板块结构（10分）

观察板块梯队：

例如：

- 龙头
- 二板
- 跟风

评分：

涨停≥5 且无跌停 → +10  
涨停≥3 → +6  
仅1-2个涨停 → +3  
无涨停 → 0

---

### 龙头反馈（10分）

判断：

板块龙头是否：

- 连板
- 高开
- 强势震荡

评分：

龙头连板 → +10  
龙头高开震荡 → +6  
龙头转弱 → -5
